# LoopusBrowser

> LoopusBrowser 当前处于 Beta 阶段。项目后续将根据开发进度逐步开放部分源代码；现阶段仅开放已经发布的二进制文件，暂不公开完整源代码。

LoopusBrowser 是一款面向 Windows 的指纹浏览器管理工具，帮助用户在同一台计算机上创建、配置和管理多个相互隔离的浏览器环境。

每个浏览器环境都可以拥有独立的浏览器配置、指纹参数、代理网络、扩展和持久化浏览数据，适合电商运营、社交媒体运营、广告团队以及需要管理多个长期浏览器身份的专业用户。

## 主要特性

- 独立的浏览器环境和持久化用户数据
- 稳定、固定种子或每次启动随机的指纹配置
- 支持 HTTP 和 SOCKS5 代理
- 代理认证、批量导入、健康检查和出口 IP 检测
- 浏览器扩展导入与环境分配
- 可见模式与无头模式
- 浏览器进程生命周期管理和异常恢复
- CloakBrowser 浏览器引擎的版本管理与安全更新
- 环境备份、恢复和身份迁移
- 应用程序 Beta / Stable 更新通道
- 发布文件哈希校验、数字签名和完整性验证
- 原生 Windows 桌面体验
